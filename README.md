## React webpack starter kit
Starter kit to provide ready-to-go webpack support for react.

## Goals
- babel + es6 support
- sass support
- hot reloading
- easy build command
	- https://webpack.js.org/guides/production-build/
- easily customisable
	- dev server port
	- path where to dump build
	- bundle name
	- proxy requests
- DONE eslint
- add lodash
